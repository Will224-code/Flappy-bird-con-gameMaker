# Flappy Bird - Game Maker [Proyecto]

Bienvenido al repositorio de **Flappy Bird**, un videojuego desarrollado en Game Maker por **@wil224-code**; Este espacio está dedicado a documentar de forma transparente todo el proceso de creación, dividido en 4 fases principales: Análisis, Diseño, Programación y Arte.

## Descripcion General

Recreacion del videojuego flappy bird, la cual tendra 3 salas, cada sala con una tarea inportante, en donde se podra seleccionar el aspecto del personaje, podra verse el puntaje obtenido y poder divertirse.

##  Resumen de Arquitectura del Juego (Rooms)

El El juego se basa en 3 salas, con un flujo lineal; Cada una de estas salas tiene un objetivo.

1. **`RM_Inicio`**: Menú principal con selección de personajes y botón de inicio.
2. **`RM_Juego`**: Pantalla de gameplay interactivo y conteo de puntaje en tiempo real.
3. **`RM_Final`**: Pantalla de Game Over con estadísticas del intento y récords personales.
